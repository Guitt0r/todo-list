# todo-list

Simple todo list write on the MERN stack

## Technology Stack:
- Node js
- Express Js
- MongoDB
- React
- React Bootstrap
- Redux

## Usage

### Env Variables

Create a .env file in server directory, which contains:

```
DB_URL='your MongoDb url'
PORT=5000
```

## Install Dependencies

```
cd server
yarn
```

```
cd ../client
yarn
```

## Run

Run client:
```
yarn start
```

Run backend:
```
yarn serve
```
